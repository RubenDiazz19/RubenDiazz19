## Sobre mí

> *"Un sistema bien diseñado no es el que no tiene nada que añadir, sino el que no tiene nada que quitar."*

Soy estudiante de **Ingeniería de Tecnologías de Telecomunicación** en la ULPGC, con una fuerte inclinación hacia los sistemas telemáticos, las redes seguras y el desarrollo de software eficiente. Me mueve la curiosidad constante: no me conformo con que algo funcione, necesito entender **por qué** funciona y cómo hacerlo mejor.

Detallista por naturaleza y analítico por formación, disfruto tanto diseñando arquitecturas de red como optimizando flujos de trabajo en el terminal. Si existe una forma más elegante de resolver un problema, la voy a encontrar.

- 🎓 **Grado en Ing. Tecnologías de Telecomunicación** · ULPGC (2021 – Actualidad)
- 🔭 Explorando sistemas distribuidos, virtualización avanzada y DevOps
- 🌱 Aprendiendo continuamente sobre arquitecturas modernas de red y nube
- 🧠 Mentalidad: primero entiende el problema, luego optimiza la solución

> ⚠️ Para que la serpiente funcione, activa el GitHub Action que la genera. Crea el archivo `.github/workflows/snake.yml` con el siguiente contenido:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: RubenDiazz19
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
